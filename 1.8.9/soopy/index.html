    SoopyAddons Alternatives¶

    SoopyAddons is a mod that is made for ChatTriggers and can no longer be developed by Soopy. This means that as time goes on more features will break or not work as intended. It also requires ChatTriggers to run the mod which can cause performance issues.

    List is a WIP but should have all features just is missing some alternatives. Feel free to make a pull request to add them.
